> Ubutntu上安装Node

[参考](https://www.cnblogs.com/guanine/p/9392411.html)

```shell
sudo apt update
sudo apt install nodejs
sudo apt install npm 
#利用n来管理版本
sudo npm install -g n
#n来下载node版本
sudo n lts 长期支持
sudo n stable 稳定版
sudo n latest 最新版
sudo n 8.4.0 直接指定版本下载
#切换版本
sudo n v12 选择你要的版本，回车确认
#升级npm
sudo npm i -g npm
```
